EGBNB - An Innovative Rental Platform

EGBNB is a modern web application inspired by Airbnb, designed to connect hosts and guests in a seamless rental experience. Built using C#, MS SQL Server, Razor, HTML, and SASS, this platform offers a robust solution for short-term rental management.

Key Features:

  • User-Friendly Interface: Developed a responsive web interface using Razor and SASS, providing a smooth user experience for both hosts and guests.
  • Robust Property Management: Utilized C# and MS SQL Server for a reliable backend, allowing hosts to create and manage property listings with images, descriptions, and secure booking options.
  • Comprehensive Search and Review System: Implemented advanced search functionality for guests to find accommodations based on location and price, along with a review system to enhance user decision-making.

This project exemplifies the power of web technologies in creating a comprehensive rental solution, emphasizing user experience and efficient property management.

NutriFit - Diet Planning and Health Tools App

NutriFit is a standalone desktop application designed to help users manage their diet and health goals effectively. Developed using Python, PyQt, and SQLite, this app provides a comprehensive set of tools for personalized diet planning and health tracking.

Key Features:

  • Personalized Diet Planning: Users can create customized meal plans based on their dietary preferences, health goals, and nutritional needs.
  • Health Tracking Tools: Integrated features for tracking daily calorie intake, macronutrients, and exercise routines, providing users with valuable insights into their health.
  • User-Friendly Interface: Built with PyQt, the application offers an intuitive interface that enhances user experience, making it easy to navigate through various functionalities.

This project showcases the potential of desktop applications in promoting healthier lifestyles through personalized diet management and user engagement.
